# Netflix Movies & TV Shows Dataset - EDA Project

## Objective

To analyze Netflix content trends using Exploratory Data Analysis (EDA) and derive meaningful business insights.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Dataset

Netflix Movies and TV Shows Dataset from Kaggle

## Project Tasks

1. Data Loading and Inspection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Insights Report

## Key Findings

* Movies dominate Netflix's content library.
* United States contributes the highest number of titles.
* India is the second-largest content producer.
* Netflix content additions peaked in 2019.
* Most content is released after 2000.
